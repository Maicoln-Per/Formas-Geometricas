Calculador Retângulo

Um retângulo é uma figura geométrica de quatro lados, onde os lados opostos são paralelos e têm o mesmo comprimento.

A fórmula para calcular a área de um retângulo é simplesmente o produto da base (lado mais longo) pela altura (lado mais curto). Se denotarmos a base como "b" e a altura como "h", então a fórmula para calcular a área é: Área do Retângulo = base * altura
